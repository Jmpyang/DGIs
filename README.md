

## ğŸ”§ Customization

### Adding Team Members

1. **Add member photo** to `images/` directory (150x150px, JPEG format)
2. **Add project image** to `projects/` directory
3. **Add resume PDF** to `resumes/` directory
4. **Update HTML files** with member information

### Modifying Styles

The project uses CSS custom properties for easy customization:

```css

}
```

## â“ Frequently Asked Questions

<details>
<summary><strong>Q: Can I use this template for my own team?</strong></summary>
<br>
A: Absolutely! This project is open source under the MIT license. Just fork the repository, customize the content with your team's information, and deploy. Don't forget to update the team member data in both HTML files and the JavaScript resume database.
</details>

<details>
<summary><strong>Q: How do I add more team members?</strong></summary>
<br>
A: Follow these steps:
1. Add member photos to the `images/` folder
2. Add project images to the `projects/` folder
3. Add resume PDFs to the `resumes/` folder
4. Update all HTML files with the new member information
5. Add the member's resume data to the JavaScript `resumeDatabase` object in `script.js`
</details>

<details>
<summary><strong>Q: Is this website SEO optimized?</strong></summary>
<br>
A: The website follows SEO best practices with semantic HTML, proper meta tags, and clean URLs. For enhanced SEO, consider adding structured data markup, sitemap.xml, and robots.txt files.
</details>

<details>
<summary><strong>Q: Can I integrate a CMS or make it dynamic?</strong></summary>
<br>
A: Yes! While the current version is static, you can integrate headless CMS solutions like Strapi, Contentful, or Sanity. You could also add a simple backend with Node.js/Express or Python/Flask for dynamic content management.
</details>

<details>
<summary><strong>Q: How do I update the color scheme?</strong></summary>
<br>
A: The website uses CSS custom properties for easy theming. Update the color variables in the `:root` selector in `styles.css`. All components will automatically use the new colors.
</details>
